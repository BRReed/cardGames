# Black Jack

## General Info

Black Jack written in Python 3.8

## Technologies
Project is created with:

* Python 3.8+

* TKinter

* PIL

## Setup
To run this project:

* Install Python and PIL

* Install project files into same directory

```
$ cd ../blackjack

$ python bj_gui.py
```
